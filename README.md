# Sistem-Rekomendasi-SMA-SMK-Surabaya
Skoolify: Sistem Rekomendasi Sekolah SMA dan SMK di Surabaya Skoolify adalah aplikasi berbasis Streamlit yang membantu calon siswa atau orang tua menemukan sekolah SMA dan SMK negeri terbaik di Surabaya berdasarkan lokasi pengguna dan tingkat kelengkapan fasilitas sekolah.

## Fitur Utama
1. Pencarian berbasis lokasi (pengguna dapat memasukkan nama jalan dan kecamatan untuk menemukan sekolah terdekat).
2. Perhitungan jarak otomatis menggunakan rumus Haversine.
3. Rekomendasi berbasis kesamaan fasilitas (sistem menghitung cosine similarity antara preferensi pengguna dan data sekolah).
4. Visualisasi peta interaktif (menampilkan lokasi pengguna dan rekomendasi sekolah).
5. Tampilan daftar rekomendasi (menampilkan 10 sekolah terbaik berdasarkan skor gabungan jarak dan kesesuaian fasilitas).
